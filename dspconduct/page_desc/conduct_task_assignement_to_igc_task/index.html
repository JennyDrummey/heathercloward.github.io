Conduct Task Assignement to IGC Task

Use this page to [Map an IGC™ Task to An Existing dspConduct™ Task](../Use_Cases/Map_a_dspConduct_Task_to_a_Task_Imported_from_IGC.htm#Map_an_IGC_Task_to_An_Existing_dspConduct_Task) .

To access this page:

  1. Select dspConduct > Design in the Navigation pane.
  2. Click the IGC Tasks icon.
  3. Click the Use Existing Tasks icon.

Field

Description

TASK ID

Displays the dspConduct™ task assigned to the IGC™ task selected on the IGC Task page.

dspConduct™ tasks that can be mapped to IGC™ tasks are either:

  • dspConduct™ tasks in the category that are of the same task type as the IGC™ task
  • Delivered dspConduct™ tasks Request Post and Request Post Workflow Notification.

PUBLISHED

If enabled, the task is published, i.e., synched into the dspConduct™ business process(es).

If an error has occurred in the framework while attempting to map the IGC™ task to the existing dspConduct™ task an error message displays and this field is unchecked. Click the log link in the error message for more information.
