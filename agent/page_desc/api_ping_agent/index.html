API Ping Agent

Use this page to [test the BackOffice IGC Connector Agent connection to DSP](../Config/Test_a_BackOffice_IGC_Agent_Connection_in_DSP.htm).

To access this page, navigate to Agent Interface > Configuration > APIs > API Ping Agent in the Navigation pane
Field Description
Ping Agent Click to check the BackOffice IGC Connector Agent connection to the DSP.
Base URL Displays the RESTServer address for the BackOffice IGC Agent. It must begin with http:// and end with /rpc. Include the IPv4 address. For example, http://253.17.35.22:8000/rpc. Update this address on the Parameters page.
