Configure dspCompose™ Parameters

dspCompose™ parameters are high-level fields used throughout the component that must be configured per installation.

To configure parameters:

  1. Select Configuration > Parameters on the Navigation pane.

  2. Click Edit.

    View the field descriptions for the Parameters page.

  3. Update parameters as needed.

  4. Click Save.
