Copy Plan Task

Use this page to [Copy Plan Tasks](Copy_Tasks1).

To access this page:

  1. Select Track > Project in the *Navigation* pane.
  2. Click Plans for a project.
  3. Click Tasks for a plan.
  4. Select a task NAME.
  5. Click Copy Plan Task.

Field

Description

Copy Plan Task

Click to copy the plan task after selecting copy options.

NOTE: A Plan Task cannot be copied to a Plan that is active. Refer to Activate and Deactivate a Plan for more information.

Copy Users

If enabled, users assigned to the task are included in the copied task.

Copy Dependencies

If enabled, task dependencies for the task are included in the copied task.

Copy Tags

If enabled, tags assigned to the task are included in the copied task.

Copy Task Rules

If enabled, Work List Event rules for the task are included in the copied task.

Copy Task Validations

If enabled, Work List Event validations for the task are included in the copied task.

Task Copy Suffix

Displays the suffix appended to the copied task. If no suffix is entered, the copied task will retain its original name.

Destination Plan ID

Select the name of the plan to which the task should be copied. If the task should remain in the current plan, select the current plan name.

Name

Displays the name of the copied task.

NOTE: If no task name is entered, no default name is assigned. Another task with the same name is created.

Description

Enter a description of the copied task.
