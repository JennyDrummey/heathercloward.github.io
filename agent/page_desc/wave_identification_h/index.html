Wave Identification H

Wave Identification V

Use this page to [Configure the Migration dashboard](../Use_Cases/Configure_the_Migration_Dashboard.htm).

To access this page, select Agent Interface > Dashboards > Migration Dashboard in the Navigation pane.

Field

Description

WAVE ID

Displays the name of the active wave. Click the wave name to open the Waves page for that wave in Console.

SEND WAVE AND MILESTONE METRICS

If checked, the wave is included on the IGC Migration dashboard.

If unchecked, the wave does not display on the dashboard.

NOTE: If the check box was checked and data has been sent to the dashboard, then the check box is unchecked, the wave still displays on the dashboard, but metrics are no longer sent.

DESCRIPTION

Displays a description of the wave as entered on the Waves page.

START DATE

Displays the date, entered on the Vertical View of the Waves page, that work on the migration for this wave begins. This date is used as a milestone by default in the Migration dashboard.

END DATE

Displays the date, entered on the Vertical View of the Waves page, that work on the migration for this wave ends. This date is used as a milestone by default in the Migration dashboard.

NOTE: if the current date is one month or later past this end date, the wave does not display on the dashboard.

Milestones

Click to open the Wave Milestones page to add milestones and thresholds to the wave that display on the dashboard.

Data Quality Report Assignment

Click to open the Data Quality Report Assignment page to set the reports used in the Data Quality metrics calculations.
