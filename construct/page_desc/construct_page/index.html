Construct Page

Use this page to [View the Construction Page in Construct](../Use_Cases/View_the_Construction_page_in_Construct).

To access this page:

  1. Select dspMigrate in the Navigation pane.
  2. Select Construct in the Context bar.
  3. Click the Objectsicon on the Construct Launch page.
  4. Click the Targets icon for an Object.
  5. Click the Construct Pages icon for the Target.

Field

Description

OBJECT NAME

Displays the name of the Object that contains Targets to which Construct pages are added.

Objects are added in Console.

OBJECT DESCRIPTION

Displays a description of the Object as added in Console.

TARGET NAME

Displays the name of the Target to which the Construct page has been added.

Targets are added in Target Design.

TARGET DESCRIPTION

Displays a description of the Target added in Target Design.

PAGE NAME

Displays the page name of the Construct page into which data is entered.

SOURCE DATA SOURCE

Displays the data source for the Construct page (by default, sdbDSPConstruct).

COMPLETED RECORDS

Displays the number of completed records for the Target. Records are complete based on client-specific requirements enforced by stored procedures registered to the Construct page.

TOTAL RECORDS

Displays the total number of fields in the source data source for the source table for the Target.

EXPECTED RECORDS

Displays the total number of estimated fields to be completed (i.e., the number of active fields in the Target).

This is added in Target Design for Full Construction types only on the vertical view of the Target Sources page for a source of {Full Construction}.

Construct Page

Click to open the Construct page used to enter data.
