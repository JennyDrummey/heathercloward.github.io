User Context Setting

Use this page to [Set the Default Context for a User](../../../Platform/Sys_Admin/Use_Cases/Set_the_Default_Context_for_a_User.htm) (performed by an Administrator) or to set the user's own context.

To access this page:

  1. Select dspMigrate in the Navigation pane, or select Console in the Context bar.
  2. Select Advanced Configuration > User Context Setting in the Navigation pane.

Field

Description

USER ID

Displays the user ID to which the default context setting is being applied.

DEFAULT Wave Process Area ID

Displays the default Wave Process Area ID for the user.

NOTE: The current user can set the default context for their own account. This setting can also be changed by the Administrator user.

NOTE: Each time the selected user logs in the DEFAULT Wave Process Area ID chosen displays in the Context bar.
