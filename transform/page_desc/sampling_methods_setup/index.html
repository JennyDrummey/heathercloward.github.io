Sampling Methods (Setup)

Use this page to [Configure Sampling Methods.](../Config/Configure_Sampling_Methods.htm)

To access this page:

  1. Access Transform.
  2. Select Configuration > Setup > Sampling Methods (Setup) in the *Navigation* pane.
Field Description
DESCRIPTION Displays the name of the sampling method to use when selecting random rows of data for verification during the audit process.
ACTIVE If checked, the sampling method is available in the Sample Method list box on the Process Information tab on the Target Report page’s Vertical View and can be used to select a sample of data from a target report to verify.
TOGGLE ACTIVE Click to set a sample method as active or inactive.
Sample Size Click to open the Sample Sizes (Setup) page to set the range and sample size to be used by the sample method.
